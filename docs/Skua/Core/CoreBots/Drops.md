## Methods

| Method Definition                   | Return Type | Description                                       |
| ----------------------------------- | :---------: | ------------------------------------------------- |
| `AddDrop(params string[] items)`    |   *void*    | Adds drops to the pickup list, un-bank the items. |
| `RemoveDrop(params string[] items)` |   *void*    | Removes drops from the pickup list.               |
